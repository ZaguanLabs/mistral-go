## Migration Path for Users

### From v2.0.1 to v2.1.0

Users need to update code that uses:

1. **Library fields**: `Created` → `CreatedAt`, `Updated` → `UpdatedAt`
2. **Document fields**: `Status` → `ProcessingStatus`, handle nullable `Size`
3. **Files API**: Handle nullable `Total` field
4. **New required fields**: Ensure API responses include new required fields

### Example Migration

```go
// Before v2.1.0
library.Created
doc.Status
if response.Total == 2 { }

// After v2.1.0
library.CreatedAt
doc.ProcessingStatus
if response.Total != nil && *response.Total == 2 { }
```
